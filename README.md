# Projeto Cordel

Página web desenvolvida como projeto prático do curso de HTML5 e CSS3, baseada no poema "Cordel Moderno" de Milton Duarte.

## Sobre o projeto

O projeto aplica técnicas de estilização avançada com CSS3 para criar uma página com visual inspirado na literatura de cordel brasileira — incluindo tipografia, cores e layout temáticos.

## Técnicas aplicadas

- Estruturação semântica com HTML5
- Estilização com CSS3 (fontes customizadas, paleta de cores temática, espaçamento e layout)
- Uso de imagens e organização de assets em pastas

## Estrutura
projeto-cordel/
├── estilo/       # Folhas de estilo CSS
├── imagens/      # Imagens utilizadas no projeto
└── index.html    # Página principal

## Como visualizar

Basta clonar o repositório e abrir o arquivo `index.html` no navegador:

```bash
git clone https://github.com/aboutgabriel/projeto-cordel.git
cd projeto-cordel
# Abra o index.html no seu navegador
```
